Config Files
============

pkg_task5/config/rviz
---------------------

It contains the scene files of the conveyor belt, the kiva pod shelf and 3 bins for the 3 types of packages.

pkg_task5/config/saved_trajectories
-----------------------------------

It has the saved trajectories of both the arms. It contains the trajectories of ur5_1 arm from home position to each box's position on the shelf.
It also has the saved trajectories of ur5_2 arm from home position to each of the three bins.
